# 資訊理論基礎：夏農信道編碼定理 與 夏農-哈特利定理

本文檔旨在說明克勞德·夏農 (Claude Shannon) 提出的兩個核心通信定理，它們共同奠定了現代數位通訊的理論基礎。

## 1. 夏農信道編碼定理 (Shannon's Channel Coding Theorem)

**又稱為「噪聲信道編碼定理」(Noisy-Channel Coding Theorem)**

這是在資訊理論中**最具突破性**的定理之一。它回答了一個在夏農之前所有人都認為是「不可能」的問題：

> **核心思想：**
>
> 只要資訊的傳輸速率（Rate, $R$）**低於**一個特定的極限——即「信道容量」(Channel Capacity, $C$)——我們就可以透過足夠精妙的編碼（例如前向錯誤更正碼, FEC），將通訊中的錯誤率**降低到任意接近於零**的程度。

換句話說，**噪聲（Noise）並不意味著錯誤（Error）是不可避免的**，而只是意味著通訊速率有一個理論上的「速限」。

### 關鍵概念

* **信道容量 (Channel Capacity, $C$)**：
    * 代表一個特定的噪聲信道，在理論上**能可靠傳輸資訊的最大速率**。
    * 這個容量 $C$ 是由信道的物理特性（例如信噪比、帶寬）所決定的。

* **傳輸速率 (Rate, $R$)**：
    * 代表我們**試圖**透過信道傳送資訊的速率（例如，每秒傳送多少 bits）。

### 定理的兩個核心論述

1.  **如果 $R < C$（速率 < 容量）：**
    * **存在**一種編碼方式，使得在接收端解碼的錯誤機率 ($\text{P}_e$) 可以被降到**任意低**（$\text{P}_e \to 0$）。
    * **意義：** 可實現「近乎完美」的可靠通訊。我們在 (7,4) 漢明碼中看到的「修正1位元錯誤」就是這個概念的簡單實踐。

2.  **如果 $R > C$（速率 > 容量）：**
    * **不存在**任何編碼方式，可以使錯誤率低於某個正的下限。
    * **意義：** 錯誤將是不可避免的，通訊將變得不可靠。

**重要性：** 這個定理是一個「存在性」定理。它石破天驚地指出了「可靠通訊的極限」是存在的，但**它沒有告訴我們 *如何* 具體設計出達到這個極限的編碼**。找到逼近 $C$ 的實用編碼（如 Turbo 碼, LDPC 碼）花了工程師們數十年的時間。

## 2. 夏農-哈特利定理 (Shannon–Hartley Theorem)

如果說「信道編碼定理」是一個**普遍的原則**，那麼「夏農-哈特利定理」就是這個原則在一個**特定且極其重要的模型**上的**具體計算公式**。

這個定理適用於「**加性高斯白雜訊 (AWGN)**」信道，這是無線電、Wi-Fi、衛星通訊等許多系統的標準模型。

### 核心公式

夏農-哈特利定理給出了 AWGN 信道的**信道容量 (C)** 的確切計算公式：

$$
C = B \log_2\left(1 + \frac{S}{N}\right)
$$

### 公式解析

* **$C$ (Channel Capacity)**：
    * 信道容量，即理論上最大的可靠傳輸速率。
    * 單位：**位元/秒 (bits per second, bps)**。

* **$B$ (Bandwidth)**：
    * 信道的帶寬（頻寬）。
    * 單位：**赫茲 (Hertz, Hz)**。
    * **意義：** 信道的「寬度」。帶寬越大，可用的「車道」越多。

* **$S$ (Signal Power)**：
    * 訊號的平均功率。
    * 單位：瓦特 (Watts)。

* **$N$ (Noise Power)**：
    * 信道中噪聲的平均功率（通常是熱噪聲）。
    * 單位：瓦特 (Watts)。

* **$S/N$ (Signal-to-Noise Ratio, SNR)**：
    * **信噪比**（或稱訊雜比）。這是整個公式中最關鍵的參數，它衡量了訊號相對於背景噪聲的「清晰度」或「強度」。
    * $S/N$ 是一個無單位的比值。在工程上，更常用**分貝 (dB)** 來表示，但計算時必須轉換回原始比值。

### 定理的啟示

夏農-哈特利公式告訴我們，要提高通訊的「速限」(C)，我們只有三種方法：

1.  **增加帶寬 ($B$)**：
    * （例如，從 20MHz 的 Wi-Fi 頻道切換到 40MHz 頻道）。
    * $C$ 與 $B$ 呈**線性**關係。帶寬加倍，容量（大致）加倍。

2.  **增加訊號功率 ($S$)**：
    * （例如，靠近 Wi-Fi 基地台，或提高發射功率）。
    * 這會提高 $S/N$。

3.  **降低噪聲功率 ($N$)**：
    * （例如，使用更好的接收器或冷卻設備）。
    * 這也會提高 $S/N$。

**關鍵的權衡 (Trade-off)：**
$C$ 與 $S/N$ 呈**對數 (log)** 關係。這意味著，當 $S/N$ 已經很高時，你必須**付出極大的功率 ($S$) 代價**，才能換取 $C$ 的一點點提升。相反，提高帶寬 $B$ 對速率的提升更為直接。

## 3. 總結：兩者的區別

| 特性 | 夏農信道編碼定理 (Channel Coding Theorem) | 夏農-哈特利定理 (Shannon–Hartley Theorem) |
| :--- | :--- | :--- |
| **類型** | **一般原則** (存在性定理) | **具體公式** (計算定理) |
| **範圍** | 適用於**任何**具有穩定特性的噪聲信道 | **僅**適用於 **AWGN** (加性高斯白雜訊) 信道 |
| **回答的問題** | 「在噪聲中，可靠通訊是否**可能**？極限在哪？」 | 「對於 AWGN 信道，這個極限**具體是多少** (bps)？」 |
| **核心關係** | $R$ (速率) vs. $C$ (容量) | $C = B \log_2(1 + S/N)$ |

**簡單來說：**

* **信道編碼定理** 說：「你開車不能超過速限 $C$，否則會出事（錯誤）。」
* **夏農-哈特利定理** 說：「在 AWGN 這條高速公路上，速限 $C$（由路寬 $B$ 和路況 $S/N$ 決定）**恰好**是 $B \log_2(1 + S/N)$ 這麼多。」
