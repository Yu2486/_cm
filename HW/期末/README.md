## 🚀 難題詳解 A：純手刻傅立葉轉換 (HW-10)

### 1. 挑戰描述
一般的訊號處理課程會直接呼叫 `numpy.fft` 或 `scipy`，但本題要求**禁止使用任何 FFT 套件**，必須回到數學定義去實作正轉換 (DFT) 與逆轉換 (IDFT)。這對於理解頻域與時域的轉換本質非常有幫助。

### 2. 數學原理
離散傅立葉轉換 (DFT) 的定義公式為：
$$X_k = \sum_{n=0}^{N-1} x_n \cdot e^{-i \frac{2\pi}{N} k n}$$

其中 $e^{-i \theta} = \cos(\theta) - i \sin(\theta)$ (歐拉公式)。

### 3. 實作細節
我使用 Python 內建的 `cmath` (複數運算庫) 與雙重迴圈來實現：
* **外層迴圈**：遍歷每一個頻率成分 $k$。
* **內層迴圈**：對所有時間點 $n$ 進行加權求和。
* **逆轉換 (IDFT)**：公式結構相似，但指數符號為正，且最後需除以 $N$。

雖然此算法的時間複雜度為 $O(N^2)$，不及快速傅立葉轉換 (FFT) 的 $O(N \log N)$ 快，但對於理解「訊號即向量內積」的概念是最佳的練習。

## 🚀 難題詳解 B：常係數齊次 ODE 通用求解器 (HW-11)

### 1. 問題描述
撰寫一個通用函數 `solve_ode(coefficients)`，輸入任意階數的微分方程係數（例如 `[1, -5, 6]` 對應 $y'' - 5y' + 6y = 0$），輸出其數學解析解。

### 2. AI 協作與解決方案
此題涉及複雜的符號邏輯判斷，我使用 **Gemini** 協助設計 SymPy 的操作流程：
* **特徵根判別**：程式能自動區分**相異實根**、**重根** (需補乘 $t^k$) 與 **共軛複數根** (轉換為 $e^{\alpha t} \cos \beta t$)。
* **符號組裝**：將判斷後的各項線性疊加，自動生成最終的通解公式。

### 3. 成果
該程式不僅能解二階電路問題，也能處理高階系統，成功將微積分問題轉化為代數問題求解。
